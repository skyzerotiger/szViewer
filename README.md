# szViewer
szViewer는 일렉트론으로 개발된 데스크탑용 이미지뷰어 입니다.   
무료이며 쉽고 빠르게 이용할 수 있는 이미지뷰어를 목표로 개발하고 있습니다.   

szViewer is an image viewer for desktop developed with Electron.   
Free, easy and quick to use is goal.
